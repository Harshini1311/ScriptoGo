ScriptoGo

ScriptoGo is a web application designed to help creators and professionals write better content faster.  
It simplifies the process of turning ideas into structured scripts and posts, especially for platforms like "YouTube" and "LinkedIn".

The goal of ScriptoGo is not to replace creativity, but to reduce friction by giving users a strong starting point that they can edit, refine, and publish confidently.

---

 What ScriptoGo Does

- Helps generate clear, structured content from a simple topic or idea  
- Supports multiple languages for wider reach  
- Allows users to save, revisit, and manage previous scripts  
- Provides a focused editor experience without distractions  

---

Key Features

- User Authentication:
  Secure login and session handling using Supabase.

- Content Dashboard:
  Central place to view and manage generated scripts.

- Editor Interface:  
  Clean and minimal editor built for iteration and clarity.

- Modern UI:
  Responsive layout built with Tailwind CSS.

---

Tech Stack

- Next.js (TypeScript)
- Tailwind CSS
- Supabase (Authentication & backend services)
- Node.js
- Netlify (Deployment)
